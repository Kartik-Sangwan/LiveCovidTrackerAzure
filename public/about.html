About

  • This is web application that tracks current location people that may have covid in your area.
  • You can check the map when you are outsied or while leaving and it will show the latest map with most recent live locations.
  • Please help spread this website, so that we can reach out to maximum number of people. Lets fight COVID together.
  • For more information on COVID government statistics visit Worlometers website.
  • Web Application created using HTML, CSS, bootstrap, JavaScript, Node.js.
  • STAY STRONG STAY SAFE


Contribute

I would like to make this project open-sourced so that we can advance it together from here and create a powerful application. The github links for the project will be uploaded soon. Please create a branch if you would like to add any new feature/modification.
